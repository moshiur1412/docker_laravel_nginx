# How to Do a Clean Restart of a Docker Instance
================================================
docker-compose down

Delete all containers:
docker rm -f $(docker ps -a -q)

Delete all volumes:
docker volume rm $(docker volume ls -q)

Restart the container:
docker-compose up -d
